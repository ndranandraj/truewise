# Truewise — Impact Log

_The hardest evidence to reconstruct later, so it is recorded from day one. Every artifact,
mention, user, download, citation, talk, and press hit gets a dated entry here. Keep dated
screenshots alongside (in `docs/impact/`) for anything that can disappear (tweets, stats
pages, articles)._

**What counts as impact (log all of it):**

- Artifacts shipped: repo public, dataset releases + DOIs, PyPI releases, write-ups, widgets.
- Adoption: GitHub stars/forks, PyPI download stats, dataset downloads, sites embedding the widget.
- Reach: press mentions, newsletter inclusions (e.g. Data Is Plural), blog cross-posts, HN/Reddit.
- Recognition: talk acceptances, citations, independent-user emails (keep the full thread).
