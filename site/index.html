Open data, not another comparison site

Truewise is open data infrastructure researchers and journalists can build on — the website is the demo layer. Everything comes from public federal sources (College Scorecard, IPEDS, Federal Student Aid), cleaned into one documented dataset with a reproducible pipeline, published under an open license.

$ pip install truewise-data Source & data on GitHub →

How we stay honest

  • Every number carries its source and cohort year
  • We never impute — thin data says “insufficient data”
  • Figures are framed as “students like you earned roughly” — never a promise
